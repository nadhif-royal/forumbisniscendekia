This repository contains the source code for the official landing page of **Forum Bisnis Cendekia (FBC)**. 

FBC is a strategic organization acting as an incubator and event organizer dedicated to developing business competencies among students in Indonesia. The website serves as the primary digital presence to showcase the organization's vision, missions, and its flagship national event, the **Future Founders League**.

## âœ¨ Key Features

* **Responsive Design:** Fully adaptive layout that ensures a seamless experience across desktop, tablet, and mobile devices.
* **Modern UI/UX:** Clean, professional interface utilizing a Navy Blue and Gold color palette to represent professionalism and prestige.
* **Interactive Elements:**
    * **Dynamic About Section:** Features interactive cards with hover effects (tilt and lift animations) to highlight core organizational pillars.
    * **Scroll Animations:** Elements fade in and slide up gracefully using the Intersection Observer API.
* **Event Showcase:** A dedicated section highlighting the *Future Founders League 2026*, complete with participation statistics and event details.
* **Contact Integration:** Direct links to email and social media platforms.
